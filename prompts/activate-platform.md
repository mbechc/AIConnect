# Activate Platform Prompt

Use this prompt when the customer asks to activate AI Connect.

Read `AGENTS.md`, `docs/agent/00_START_HERE.md`, and `runbooks/activate-platform.md`.

Ask first:

```text
Is this a lab, pilot, or production activation?
```

Proceed one step at a time. Do not request secret values. Ask for approval before changing infrastructure, production configuration, credentials, DNS, certificates, customer data, or device state.
